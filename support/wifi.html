WiFi Configuration

If you encountered with product configuration issues, please pay attention to the following checklist:

1. Please make sure you are using our genuine product with official App “Intelligent Home Center” (ihc).

2. Our products support only 2.4G Wi-Fi (does not support 5G). If your home router works in dual bands (both 2.4G and 5G), please switch off 5G signal and try again. After configuration is complete, you can switch back to use dual bands (as shown below).

3. If the device Wi-Fi indicator is not off, that means the device cannot join any network. To fix this, you may need to try the following actions:

a. Reset the device (Wi-Fi indicator flashes quickly)

b. Check the Wi-Fi name. It cannot contain space or emoji symbols

c. Check the Wi-Fi password. It should be not more than 32 characters and cannot contain space or emoji symbols

d. Recommend to disable wireless MAC filtering and enable WMM on router (as shown below)

e. Do not use IP binding

f. Check if you have too many devices connected to your router (normally a home router can hold no more than 10 devices including smartphones, laptops, smart TVs, IP cameras….)

g. Try to change a channel (as shown below)

h. Login to the router to check if the device is connected
